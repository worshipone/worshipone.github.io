1. 通过修改注册表的方式实现自启动

1.1 实验要求

编写代码，编辑注册表的Run/RunOnce/RunOnceEx键（任选其一，并明确三个键的区别），达到让某一程序在系统启动后自动运行的目的（可以以计算器、记事本等作为目标程序）。

1.2 实验环境

Windows 7或Windows 10主机（虚拟机）；

代码编辑器；

C/C++代码运行所需环境。

1.3 实验目的

了解恶意代码自启动常用手段。

1.4 实验过程
1.4.1 实验思路

  1. 打开注册表

    RegOpenKeyEx(HKEY_LOCAL_METHINE\SOFTWARE\Microsoft \Windows\CurrentVersion\Run)

  2. 添加键值

    键值为要启动程序的路径(可通过GetModuleFileName获得程序路径)

  3. 关闭注册表

    RegCloseKey

1.4.2 C语言代码
#include <windows.h>
#include <iostream>
#include <cstdlib>
using namespace std;

int main()
{
    HKEY hKey;
    LPCTSTR lpRun = "Software\\Microsoft\\Windows\\CurrentVersion\\Run";
    DWORD state,dwtype,sizeBuff;
    long lRet;
    char reBuff[100] = {0};


    //创建键
    lRet = RegCreateKeyEx(HKEY_CURRENT_USER,lpRun,0,NULL,0,0,NULL,&hKey,&state);
    if(lRet == ERROR_SUCCESS)
    {
        if(state == REG_CREATED_NEW_KEY)
            cout<<"表项创建成功"<<endl;
        RegCloseKey(hKey);
    }
    //创建或者修改键值
    lRet= RegOpenKeyEx(HKEY_CURRENT_USER, lpRun, 0, KEY_WRITE, &hKey);
    if(lRet == ERROR_SUCCESS)
    {
        RegSetValueEx(hKey, "QQ",0,REG_SZ,(BYTE *)"D:\\QQ\\Bin\\QQScLauncher.exe",100);
        RegCloseKey(hKey);
    }
    //读取键值
    lRet= RegOpenKeyEx(HKEY_CURRENT_USER, lpRun, 0, KEY_READ, &hKey);
    if(lRet == ERROR_SUCCESS)
    {
        sizeBuff = sizeof(reBuff);
        if(RegQueryValueEx(hKey,"QQ",0,&dwtype,(BYTE*)reBuff,&sizeBuff) == ERROR_SUCCESS){
            cout<<"读取QQ的键值"<<endl;
            cout<<reBuff<<endl;
        }
        RegCloseKey(hKey);
    }
    system("pause");
}

3. 简单多线程服务器

3.1 实验要求

编写一个简单的echo服务器程序（即：客户端与服务器建立连接后，在客户端输入消息，服务器端就会打印输入的消息），在4444端口进行监听。每有一个客户端进行连接时候，服务器创建一个子线程，对客户端程序进行服务。根据要求，客户端代码也需要自行编写。

3.2 实验环境

Windows 7或Windows 10主机（虚拟机）至少两台；

代码编辑器；

C/C++代码运行所需环境。

3.3 实验目的

了解恶意代码在通信时，会使用到的最基本的技术。练习多线程编程。
